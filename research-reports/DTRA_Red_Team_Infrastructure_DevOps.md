## Scope of Work

The contractor will provide comprehensive infrastructure and DevOps support for the **Red Team Shared Infrastructure (RTSI)**, which supports DTRA's Cyber Red Team Assessment Operations. The work includes:

### Infrastructure Operations
- Operate, maintain, and defend RTSI end-to-end
- System and network infrastructure management
- Integrations with external operational cloud infrastructure
- Ensure reliability, scalability, and security of all RTSI components
- Compliance with DoD, DTRA, and DCART policies

### Security & Compliance
- Implement security monitoring and updates
- Assess security controls
- Develop and maintain backup and disaster recovery plans
- Mitigate power loss and hardware failure scenarios
- DoD/DTRA/DCART policy compliance enforcement

### DevOps & Automation
- **Infrastructure as Code (IaC)** implementation and maintenance
- Automation solutions for security enforcement
- Observability and logging across the environment
- Network traffic monitoring

### Dashboard & Monitoring
- Develop and maintain a functional infrastructure dashboard
- Real-time system performance monitoring
- Security posture visibility

## Competitive Positioning Notes

### What Makes This Opportunity Interesting
1. **Full & open** — no set-aside, so any size business can bid (but may benefit from small business teaming)
2. **No identified incumbent** — could be a new requirement or a contract structure change, reducing incumbency advantage
3. **Cyber Red Team** context — this is offensive security infrastructure, not standard IT. Requires cleared DevOps talent with red team/offensive security understanding
4. **IaC and automation emphasis** — DTRA wants modern practices, not legacy admin. Favors companies with strong DevSecOps delivery

### Who's Likely Competing
- Large defense IT primes: Leidos, Booz Allen, ManTech, SAIC, Peraton
- Mid-tier cyber firms: Parsons, ECS Federal, CACI
- Niche cyber/red team shops: Cobalt Strike ecosystem vendors, offensive security specialists
- Small business teams leveraging NAICS 541512 size standard ($34M)

### Risks
- **Clearance requirements** — Red Team operations almost certainly require TS/SCI, limiting the talent pool
- **Full & open** — small businesses compete against primes without set-aside protection
- **Scope complexity** — combines infrastructure ops, security, DevOps, and cloud integration in one contract
