<task type="auto">
  <name>Task 2: Create Tier 1 validation library</name>
  <files>lib/validation/tier1-validators.ts</files>
  <action>
    Create `lib/validation/tier1-validators.ts` exporting these pure validation functions:

    **validateUEI(uei: string): {valid: boolean, error?: string}**
    - Must be exactly 12 characters
    - Must be alphanumeric (letters and digits only)
    - First character must be a letter (per SAM.gov UEI format)
    - Must not contain O or I (per SAM.gov rules — UEI uses 0/1 instead of O/I to avoid confusion)
    - Return descriptive error messages: "UEI must be exactly 12 characters", "UEI must start with a letter", "UEI cannot contain letters O or I (use 0 and 1 instead)"

    **validateNAICS(code: string): {valid: boolean, error?: string}**
    - Must be exactly 6 digits (string of digits, not a number)
    - Must start with 11-92 (valid NAICS sector ranges: 11 Agriculture through 92 Public Admin)
    - Return: "NAICS code must be exactly 6 digits", "Invalid NAICS sector code"

    **validateCAGE(cage: string): {valid: boolean, error?: string}**
    - Must be exactly 5 alphanumeric characters
    - Return descriptive error

    **validateWordCount(text: string, min: number, max: number): {valid: boolean, wordCount: number, error?: string}**
    - Count words by splitting on whitespace
    - Return current word count and whether it's within bounds
    - Error messages: "Minimum {min} words required (currently {count})", "Maximum {max} words exceeded (currently {count})"

    **checkSAMStatus(uei: string): Promise<{registered: boolean, status: string, expiration?: string, error?: string}>**
    - For now, implement as a MOCK that returns a plausible response based on UEI format validity
    - If UEI is valid format: return {registered: true, status: 'Active', expiration: one year from now}
    - If UEI is invalid format: return {registered: false, status: 'Not Found'}
    - Add a TODO comment: "Replace with real SAM.gov Entity API (api.sam.gov) integration when API key is available"
    - This allows the UI to show SAM.gov cross-reference status without blocking on external API access

    All functions should be pure (no side effects, no imports needed except for checkSAMStatus which is async). Use strict TypeScript with explicit return types.
  </action>
