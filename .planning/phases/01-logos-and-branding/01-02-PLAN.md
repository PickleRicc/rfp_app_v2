**Required exports:**

1. `createProposalHeader(logoBuffer: Buffer | null, volumeTitle: string, solicitationNumber: string): Header`

   - If logoBuffer is null: Skip logo (graceful fallback)
   - Volume title center-ish
   - Solicitation number on right

2. `createProposalFooter(): Footer`
   - Left: "PROPRIETARY - Use or disclosure subject to restrictions" in small italic
   - Right: "Page X of Y" using PageNumber.CURRENT and PageNumber.TOTAL_PAGES
   - Use tab stops: left (0), right (9000) twips

3. `createEmptyHeader(): Header`
   - Returns Header with empty Paragraph
   - Used for cover page (first page)

4. `createEmptyFooter(): Footer`
   - Returns Footer with empty Paragraph
   - Used for cover page (first page)

**Required imports from docx:**
```typescript
import {
  Header,
  Footer,
  Paragraph,
  TextRun,
  ImageRun,
  PageNumber,
  AlignmentType,
  TabStopType,
  TabStopPosition,
} from 'docx';
```

**Implementation notes:**
- Import HEADER_LOGO_SIZE from './images' for consistent sizing
- Use TabStopType.RIGHT for right-aligned elements
- Proprietary notice color: Use a subtle gray like '64748b' per research
- Font: Arial, size 18 (9pt) for header/footer text
- Include JSDoc documentation for all exports

**Example footer structure:**
```typescript
new Paragraph({
  children: [
    new TextRun({
      text: 'PROPRIETARY - Use or disclosure subject to restrictions',
      italics: true,
      size: 16, // 8pt - small
      color: '64748b',
    }),
    new TextRun({ text: '\t' }), // Tab to right position
    new TextRun({ text: 'Page ', size: 18 }),
    new TextRun({ children: [PageNumber.CURRENT], size: 18 }),
    new TextRun({ text: ' of ', size: 18 }),
    new TextRun({ children: [PageNumber.TOTAL_PAGES], size: 18 }),
  ],
  tabStops: [
    { type: TabStopType.RIGHT, position: 9000 },
  ],
})
```
