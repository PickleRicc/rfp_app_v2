The logic must:

1. **Mark base document(s) as superseded when amendments have `supersedes_section` change_type records:**
   - Filter `results` for entries with `change_type === 'supersedes_section'`
   - If any exist, the base document (`baseDoc`) has been at least partially superseded by the latest amendment
   - UPDATE `solicitation_documents` SET `is_superseded = true`, `superseded_by = {latest_amendment_id}` WHERE `id = baseDoc.id`
   - The latest amendment is `amendments[amendments.length - 1]` (already sorted chronologically)

2. **Mark earlier amendments as superseded when a newer amendment exists:**
   - If `amendments.length > 1`, all amendments except the last one are superseded
   - For each amendment except the last: UPDATE `solicitation_documents` SET `is_superseded = true`, `superseded_by = {next_amendment_id}` WHERE `id = amendment.id`
   - The `superseded_by` for amendment N should point to amendment N+1 (the one that replaced it)

Place this logic AFTER the successful reconciliation insert (after `totalRecords = results.length;` on line 223) and BEFORE the "Mark amendment docs as reconciled" loop (line 228).

Use the existing `supabase` client already in scope. Each update should also set `updated_at: new Date().toISOString()`.

Do NOT modify any other steps or any UI code. The UI already reads `is_superseded` and `superseded_by` correctly — it just needs the data to be written.

Example code structure:
```typescript
// Mark base document as superseded if any supersedes_section changes exist
const hasSupersededSections = results.some(r => r.change_type === 'supersedes_section');
const latestAmendment = amendments[amendments.length - 1];

if (hasSupersededSections && latestAmendment) {
  await supabase
    .from('solicitation_documents')
    .update({
      is_superseded: true,
      superseded_by: latestAmendment.id,
      updated_at: new Date().toISOString(),
    })
    .eq('id', baseDoc.id);
}

// Mark earlier amendments as superseded by their successor
if (amendments.length > 1) {
  for (let i = 0; i < amendments.length - 1; i++) {
    await supabase
      .from('solicitation_documents')
      .update({
        is_superseded: true,
        superseded_by: amendments[i + 1].id,
        updated_at: new Date().toISOString(),
      })
      .eq('id', amendments[i].id);
  }
}
```
