**Create function estimateSectionPages:**
```typescript
export function estimateSectionPages(
  sectionContent: string | Paragraph[],
  hasExhibits: boolean
): number
```
- If string content: estimate from character count / CHARS_PER_PAGE
- If Paragraph array: count paragraphs, estimate ~3 paragraphs per page
- Add 0.5 page if hasExhibits
- Round up to nearest 0.5

**Create function calculateSectionPageNumbers:**
```typescript
export function calculateSectionPageNumbers(
  sections: Array<{ title: string; content: any; hasExhibits?: boolean }>
): Map<string, number>
```
- Start from COVER_PAGES + TOC_PAGES + COVER_LETTER_PAGES + 1
- For each section, calculate cumulative page number
- Return Map of section title to starting page number

**Create function getSectionPageNumber:**
```typescript

  sectionTitle: string,
  pageMap: Map<string, number>
): number | null
```
- Fuzzy match section title to find page number
- Return null if no match found
