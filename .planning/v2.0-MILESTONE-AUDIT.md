## Cross-Phase Integration

### Integration Gaps (3 found)

**1. Missing Tier 1 gate on /api/generate-proposal (HIGH)**
- `app/api/generate-proposal/route.ts` triggers Stage 2 proposal generation without checking `tier1_complete`
- Allows proposal generation for companies with incomplete Tier 1 profiles
- Affects: TIER1-05, TIER1-06

**2. enterprise_win_themes not consumed by Stage 2 (HIGH)**
- Phase 5 collects `enterprise_win_themes` (string array on company_profiles)
- Stage 2 `integrateWinThemes()` reads `value_propositions` table instead
- Users fill out win themes in Tier 1 but those themes never appear in proposals
- Affects: TIER1-04, TIER1-05

**3. Tier 1 narrative fields not consumed by templates (HIGH)**
- `corporate_overview`, `core_services_summary`, `key_differentiators_summary`, `standard_management_approach` — all collected, validated, scored, and stored
- No generation template references any of these fields
- Templates generate content from scratch via Claude AI instead of using stored boilerplate
- Affects: TIER1-04, TIER1-05
